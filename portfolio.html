在庫管理システム

業務効率化を目的としたCRUD機能付き管理アプリケーション

概要

在庫管理業務を想定し、商品データの登録・編集・削除・検索機能を実装しました。 データはlocalStorageに保存し、ブラウザ上で管理できる仕組みを構築しています。

制作背景

手作業や表計算ソフトで管理されがちな在庫業務を効率化することを目的に制作しました。 データの一元管理と検索性の向上を意識して設計しています。

使用技術

  • HTML
  • CSS
  • JavaScript
  • localStorage

実装機能

  • 商品登録
  • 一覧表示
  • 編集機能
  • 削除機能
  • 検索機能
  • データのローカル保存

工夫した点

  • 配列によるデータの一元管理
  • 再描画関数によるUI更新の統一
  • イベントリスナーによる安全なイベント管理
  • createElement + textContent によるXSS対策

今後の改善

  • ログイン認証機能の追加
  • CSV出力機能の実装
  • バックエンド連携によるAPI化
