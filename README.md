# 自治体向け 商品管理システム

## ■ 概要
自治体の返礼品管理業務を想定し、
商品登録・編集・削除・検索が可能な管理画面を制作しました。

## ■ 制作背景
地域DXに興味を持ち、業務効率化をテーマに
管理画面の基礎機能を実装しました。

## ■ 使用技術
- HTML
- CSS
- JavaScript
- localStorage

## ■ 実装機能
- 商品登録
- 一覧表示
- 編集機能
- 削除機能
- 検索機能
- データ保存機能（localStorage）

## ■ 工夫した点
- 配列でデータ管理
- 再描画関数でUI更新を統一
- イベント委譲を用いた削除処理

## ■ 今後の改善
- ログイン機能
- CSV出力機能
- バックエンド連携（API化）
