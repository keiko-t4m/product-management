# 商品管理システム（ポートフォリオ）

## ■ 概要
JavaScriptで開発した商品管理アプリケーションです。  
商品の登録・編集・削除・検索機能を実装し、  
localStorageを使用してデータの永続化を行っています。

GitHub Pagesで公開しています。

(https://keiko-t4m.github.io/steamship-product-management/)

## ■ 主な機能

- 商品の新規登録
- 編集・削除（CRUD機能）
- リアルタイム検索機能
- 半角数字バリデーション
- Enterキーでの登録操作
- localStorageによるデータ保存
- 一意IDによるデータ管理
- IDの5桁フォーマット表示（padStart使用）

## ■ 設計の工夫

### ① index管理から一意ID管理へ変更
初期は配列のindexでデータを管理していましたが、
削除や検索時にデータの整合性が崩れる可能性があったため、
一意IDによる管理方式へ変更しました。

### ② 連番IDの永続管理
削除によるID重複を防ぐため、
`nextId` を localStorage に保存し、
常に一意な連番IDを生成する設計にしています。

### ③ 表示と内部データの分離
内部では数値IDを保持し、
表示時のみ `padStart()` を使用して
5桁フォーマットで表示しています。

### ④ render関数による再描画設計
データ変更後は必ず `render()` を実行し、
画面表示とデータ状態の整合性を保つ構造にしています。
